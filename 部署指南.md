# 🍅 番茄钟 — 华为手机部署与调试指南

## 一、前置说明

PWA（渐进式 Web 应用）安装到手机后：
- 📱 桌面显示独立图标，点击即启动
- 🔌 离线可用（已缓存核心资源）
- 🖥 全屏运行，无浏览器地址栏
- 🔔 支持提示音循环播放

---

## 二、部署方案：GitHub Pages（免费，推荐）

### 2.1 准备工作

1. 注册 GitHub 账号：https://github.com
2. 电脑安装 Git：https://git-scm.com/download/win

### 2.2 推送代码到 GitHub

```bash
# 在项目目录执行
cd "D:\ClaudeOutput\PomodoroTimer mobile vision\mobile-pwa"

# 初始化 Git 仓库
git init
git add .
git commit -m "[feat] 番茄钟 PWA 移动版"

# 创建 GitHub 仓库并推送（替换 Cathy729 为你的用户名）
git remote add origin https://github.com/Cathy729/pomodoro-timer.git
git branch -M main
git push -u origin main
```

### 2.3 开启 GitHub Pages

1. 打开仓库页面 → **Settings** → **Pages**
2. **Source**: `Deploy from a branch`
3. **Branch**: `main`，文件夹选 `/ (root)`
4. 点击 **Save**
5. 等待 1-2 分钟，获得网址：`https://Cathy729.github.io/pomodoro-timer`

### 2.4 验证部署

电脑浏览器打开 `https://你的用户名.github.io/pomodoro-timer`，应能看到番茄钟界面。

## 三、华为手机安装（像普通 App 一样）

### 步骤 1：打开网址

在华为手机浏览器（或 Chrome）中打开上述 GitHub Pages 网址。

### 步骤 2：添加到主屏幕

| 浏览器 | 操作 |
|--------|------|
| **华为浏览器** | 底部菜单 → 「添加到桌面」或 右上角 `…` → 「添加到主屏幕」 |
| **Chrome** | 右上角 `⋮` → 「添加到主屏幕」→ 「安装」 |
| **Edge** | 底部 `…` → 「添加到手机」 |

### 步骤 3：确认安装

弹出安装提示 → 点击「添加」/「安装」。桌面出现 🍅 番茄钟图标。

### 步骤 4：首次使用

1. 点击桌面图标启动（全屏打开）
2. 编辑时段序列，选择提示音
3. 点「▶ 开始」计时
4. 即使断网也能正常使用 ✅

## 四、华为手机调试流程

### 4.1 调试模式（Chrome 推荐）

1. 华为手机安装 **Chrome 浏览器**
2. 电脑 Chrome 打开 `chrome://inspect`
3. 手机 USB 连接电脑，开启「开发者选项」→「USB 调试」
4. 手机 Chrome 打开番茄钟网页
5. 电脑 `chrome://inspect` 中看到设备 → 点击 **inspect**
6. 即可在电脑上调试手机页面的 Console / Network / Elements

### 4.2 常见问题排查

| 现象 | 可能原因 | 解决 |
|------|----------|------|
| 网页能打开但无法安装 | 不是 HTTPS | GitHub Pages 自动 HTTPS，确保用 `https://` |
| 安装后图标是白的 | 图标路径错误 | 确认 `icons/icon-192.png` 和 `icon-512.png` 存在 |
| 提示音不响 | 音频文件太大未加载完 | 等待几秒，或用较小的提示音文件 |
| 离线打开空白 | Service Worker 未激活 | 首次需联网打开，之后自动缓存 |
| 计时不准（锁屏后暂停） | 系统省电限制 | 华为：设置 → 应用 → 番茄钟 → 电池 → 允许后台活动 |
| 音频文件无法播放 | MP3/WAV 兼容性 | 已同时提供 MP3 和 WAV，华为浏览器均支持 |

### 4.3 清除缓存重新加载

如果更新了代码：
1. 手机浏览器打开番茄钟
2. 浏览器设置 → 清除该网站数据
3. 重新打开即可获取最新版本

## 五、部署方案 2：本地网络（无需上传）

如果不想公开到 GitHub，可在电脑启本地服务器，手机连同一 WiFi 访问：

### 5.1 启动本地服务器

```bash
# 进入 PWA 目录
cd "D:\ClaudeOutput\PomodoroTimer mobile vision\mobile-pwa"

# Python 方式
python -m http.server 8080

# 或 Node.js 方式
npx serve .
```

### 5.2 手机访问

1. 查看电脑 IP：命令行输入 `ipconfig`，找到 IPv4 地址（如 `192.168.1.5`）
2. 手机浏览器打开 `http://192.168.1.5:8080`
3. **注意**：HTTP 下 PWA 安装可能受限，建议用方案 1 正式部署
