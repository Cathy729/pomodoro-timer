## 一、手机端快速安装（2 分钟）

### 第 1 步：打开网址

华为手机上用**浏览器**（推荐 Chrome）打开：

```
https://cathy729.github.io/pomodoro-timer/
```

### 第 2 步：添加到主屏幕

| 浏览器 | 操作路径 |
|--------|----------|
| **Chrome** | 右上角 `⋮` → 「添加到主屏幕」→ 弹出对话框点「安装」 |
| **华为浏览器** | 底部中间 `…` → 「添加到桌面」→ 确认 |
| **Edge** | 底部 `…` → 「添加到手机」→ 「添加」 |

### 第 3 步：桌面启动

桌面出现 🍅 图标，点击即全屏启动，和普通 App 一模一样。

> ✅ 不需要应用商店
> ✅ 不占用大量存储（图标只是快捷方式）
> ✅ 离线可用（首次联网打开后自动缓存）

## 二、悬浮窗（画中画）设置

计时开始后，点击「📌 悬浮窗」按钮，计时器缩成小窗浮在其他 app 上方。

### 开启权限（仅需一次）

如果悬浮窗无法启动，按以下路径设置：

```
华为手机：设置
  → 应用
    → 应用管理
      → Chrome（或你使用的浏览器）
        → 权限
          → 悬浮窗 → 选择「允许」
```

或者：

```
设置 → 应用 → 权限管理 → 悬浮窗 → 找到浏览器 → 允许
```

### 使用方式

1. 点「▶ 开始」启动计时
2. 点「📌 悬浮窗」→ 计时器缩小成画中画浮窗
3. 切换到其他 app（微信、笔记等），浮窗**始终悬停显示倒计时**
4. 点击浮窗可返回番茄钟主页面
5. 再次点击「📌 关闭悬浮」可关闭浮窗

> 浮窗显示内容：🍅/☕ 图标 + 阶段名 + 倒计时 + 进度条 + 当前第几段

## 三、功能一览

| 功能 | 说明 |
|------|------|
| ⏱ 计时序列 | 多条工作/休息时段，从上到下依次执行 |

## 四、调试与验证

### 4.1 电脑远程调试手机

```
1. 华为手机：设置 → 关于手机 → 连续点击「版本号」7 次开启开发者模式
2. 设置 → 系统和更新 → 开发人员选项 → 开启「USB 调试」
3. USB 数据线连接电脑
4. 电脑 Chrome 打开 chrome://inspect
5. 手机 Chrome 打开番茄钟网页
6. 电脑 chrome://inspect 中看到设备 → 点击 inspect
```

### 4.2 验证清单

| 检查项 | 预期结果 | 如何验证 |
|--------|----------|----------|
| 图标安装 | 桌面出现圆角 🍅 图标 | 添加到主屏幕后查看桌面 |
| 全屏启动 | 无浏览器地址栏 | 点击桌面图标启动 |
| 计时功能 | 倒计时正常走动 | 点 ▶ 开始观察 |
| 提示音 | 计时结束循环播放 | 设 1 分钟番茄测试 |
| 提示音选择 | 下拉框显示 6 项带时长 | 点击下拉查看 |
| 悬浮窗 | 画中画浮在其他 app 上 | 计时中点 📌 悬浮窗 → 切到桌面 |
| 离线使用 | 断网仍可打开 | 开飞行模式后点桌面图标 |
| 保存/加载 | 预设持久化 | 保存 → 刷新页面 → 加载 |

### 4.3 常见问题

| 现象 | 原因 | 解决 |
|------|------|------|
| 打不开网页 | GitHub Pages 被墙或 DNS 问题 | 切换 WiFi 或使用 Chrome |
| 无法安装到桌面 | 非 HTTPS 访问 | 确认 URL 是 `https://` |
| 悬浮窗按钮灰色 | 还没开始计时 | 先点「▶ 开始」 |
| 悬浮窗点不了 | 浏览器不支持 PiP | 换 Chrome 最新版 |
| 悬浮窗闪退 | 权限未开启 | 按第二章设置悬浮窗权限 |
| 锁屏后计时暂停 | 省电模式限制后台 | 设置 → 应用 → Chrome → 电池 → 允许后台 |
| 提示音不响 | 音频文件大未加载完 | 等几秒重试，或用提示音 2/3（只有 1~2 秒） |
| 图标不是圆角 | 浏览器图标缓存 | 清除浏览器缓存后重新添加到桌面 |

## 六、技术架构

```
PWA（渐进式 Web 应用）
├── HTML5 + CSS3 + Vanilla JS（零依赖，无框架）
├── Canvas + MediaStream → Picture-in-Picture API（悬浮窗）
├── HTML5 Audio API（提示音循环播放）
├── Service Worker（离线缓存）
├── Web App Manifest（全屏安装）
└── localStorage（预设持久化）
```

**兼容性**：Chrome 90+、Edge 90+、华为浏览器（Chromium 内核）
